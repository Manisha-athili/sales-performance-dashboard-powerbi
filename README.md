🔹 Project Title

Sales Performance Dashboard (Power BI)

🔹 Overview

This project analyzes sales pipeline performance using Power BI. It provides insights into revenue trends, deal conversion, and sales agent performance.

🔹 Key Metrics
Total Revenue: 10M
Total Deals: 8800
Win Rate: 48%
Avg Deal Size: 2.36K


🔹 Features
Revenue breakdown by product and sales agent
Sales funnel analysis to identify conversion drop-offs
Monthly revenue trend visualization
Performance comparison across sales agents

🔹 Key Insights
GTXPro contributes highest share of revenue (~40%+)
Significant drop-off observed at Engaging stage
Top sales agents drive majority of revenue

🔹 Tools Used
Power BI
DAX
Excel
